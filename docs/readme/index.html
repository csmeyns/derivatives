    README

    Derivatives is an exploration of all things original and copy.

    Articles about origins and originality, published here in derivative form, constitute the bulk of this book. Each of these articles has already been published under an open access (CC-BY) license elsewhere. Original publications have been stated on each article.

    The aim of this book is to ponder about text reuse, origins and copying—and why these things appear to bother people so much.

    This book is meant to be interesting and useful to both casual readers and somewhat more experienced copyists. It offers something for everyone to pick and choose from.

    Do not try to read it from front to back. Read the chapters or content you are curious about and flip back and forth as you see fit.

    This book is open, completely free to download and read. It is moreover free for anyone to comment on, and available for everyone to contribute to and help out with.

    Send your comments, pull requests or critiques and improve this book accordingly.

    This book will never be finished. Derivations will continue and so there will always be things to update in the book as well.

    This book project started at the end of September 2021.
